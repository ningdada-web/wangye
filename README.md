# vue-manage
这是一个使用elementUI搭建的后台管理系统，封装了echats柱状图、折线图的横纵坐标方便复用，二次封装axios方便使用，使用vuex对cookie与token进行记录，实现了登录登出、数据可视化、权限管理、用户管理功能，使用导航懒加载、路由前置守卫优化性能。
